## 집에서 시작하기
  1. git clone https://github.com/Jayinsightfactory/mindmap-viewer.git
     또는 기존 clone이 있으면: git pull origin main

  2. cd mindmap-viewer && npm install

  3. 로컬 실행: npm run dev  (http://localhost:4747)

  4. orbit-config 설정 (홈 디렉토리에):
     파일: ~/.orbit-config.json
     내용:
     {
       "serverUrl": "https://orbit3d-production.up.railway.app",
       "token": "orbit_1f9ff8879e95cdeaaf57c550b0b59c825531ef3c7d35ac56",
       "userId": "MMDFB0XN5580E306A1"
     }

  5. Railway 데이터 동기화:
     node bin/sync-to-railway.js --limit=2000

## 주요 파일 구조
  server.js                    - Express + WebSocket 메인 서버
  src/save-turn.js             - Claude Code 훅 수신기
  src/db.js                    - SQLite DB 모듈
  src/auth.js                  - Google OAuth + 로컬 인증
  routes/graph.js              - 그래프/세션/통계 API
  routes/workspace.js          - 워크스페이스 관리 API
  routes/follow.js             - 팔로우 시스템 API
  bin/sync-to-railway.js       - 로컬→Railway 동기화
  public/orbit3d.html          - 3D 우주 시각화 메인
  public/js/orbit3d-scene.js   - 씬 빌드 (프로젝트 클러스터)
  public/js/orbit3d-render.js  - Canvas2D 오버레이 렌더링
  public/js/orbit3d-animate.js - 인터랙션/클릭 핸들러
  public/js/orbit3d-ws.js      - WebSocket 실시간 업데이트
