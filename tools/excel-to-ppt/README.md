# 엑셀 → PPT 품목 카탈로그 생성기 v2

엑셀(.xlsx)의 **품목명 + 임베드 이미지**를 추출해, **슬라이드마다 가로×세로(그리드)를 따로 정하고** 채울 품목을 골라 PPT를 자동 생성하는 파이썬 데스크톱 앱.

## 빠른 시작 (Windows)

```powershell
python -m pip install openpyxl python-pptx pillow
python excel_to_ppt.py
```

## 바탕화면 실행파일 만들기 (콘솔창 없음)

같은 폴더의 **`build.bat` 더블클릭** →
- PyInstaller로 `--noconsole` exe 빌드 → **바탕화면에 `엑셀PPT생성기.exe`** 복사
- (빌드 실패 시) 콘솔 없는 `엑셀PPT생성기.vbs` 바로가기를 바탕화면에 생성

이후 바탕화면 아이콘만 더블클릭하면 실행되며 cmd 창은 뜨지 않습니다.

## 화면 구성 / 사용 순서

```
[엑셀 불러오기] [시트▼] [텍스트] [글자pt] [제목] ............ [PPT 내보내기]
┌─────────────────────┬──────┬──────────────────────────┐
│ 품목 라이브러리       │ 추가  │ 슬라이드 구성              │
│ ☑[썸네일] 품목명 …   │  ▶   │ ＋슬라이드 추가 −삭제 ▲▼  │
│ (드래그=복수선택)     │  ◀   │ 슬라이드1 (3×2) 4/6       │
│ 큰 체크박스(4배)      │      │ 가로[3] 세로[2]           │
│                      │      │ 들어간 품목 ▲▼           │
└─────────────────────┴──────┴──────────────────────────┘
```

1. **📂 엑셀 불러오기** → 시트 선택. 품목 라이브러리에 썸네일 + 큰 체크박스로 표시.
2. **＋ 슬라이드 추가** → 그 슬라이드의 **가로(열) × 세로(행)** 지정 (예 3×2=6칸).
3. 라이브러리에서 **드래그로 여러 품목 선택**(클릭=체크, 드래그=범위) → **선택 품목 ▶ 현재 슬라이드에 추가**.
4. 다음 슬라이드도 **＋추가 → 그리드 지정 → 품목 선택**… 반복. 슬라이드마다 개수·구성이 독립적.
5. 슬라이드 순서(▲▼), 슬라이드 내 품목 순서(▲▼) 조정.
6. **💾 PPT 내보내기** → 모든 슬라이드를 한 번에 생성.

## 자동 저장 / 복원

- 품목·이미지·슬라이드 구성은 **변경할 때마다 자동 저장**됩니다
  (`사용자폴더\.orbit_ppt\project.json`, 이미지 캐시 `\.orbit_ppt\cache\`).
- 다음 실행 시 **이전 작업이 그대로 복원**됩니다. 엑셀 파일이 없어도 캐시된 이미지로 복원.

## 그리드 / 위치

- 슬라이드: 27.5 × 19.1 cm. 가로·세로 1~6 자유 지정(셀마다 이미지 정사각 + 하단 텍스트, 비율 유지·중앙 정렬).
- 텍스트 내용 옵션: `name`(품목명) / `origin`(+원산지) / `full`(+시즌).

## 구조 (GUI 없이 단독 호출 가능)

- `load_items(path, sheet)` — 품목·이미지 추출
- `grid_slots(cols, rows)` — 가변 그리드 좌표
- `build_pptx(slides, store, out_path, ...)` — 다중 슬라이드 PPT 생성
- `save_project / load_project` — 작업 자동 저장·복원
- `run_gui()` — tkinter UI
