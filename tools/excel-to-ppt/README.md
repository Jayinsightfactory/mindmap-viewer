# 엑셀 → PPT 품목 카탈로그 생성기

엑셀(.xlsx)의 품목명 + 임베드 이미지를 추출해, PPT 템플릿과 **동일한 고정 위치(3열×2행, 슬라이드당 6품목)** 로 자동 배치하는 파이썬 데스크톱 앱.

## 설치

```bash
pip install openpyxl python-pptx pillow
```

## 실행

```bash
python excel_to_ppt.py
```

## 사용 순서

1. **📂 엑셀 불러오기** — `.xlsx` 선택. `품목` 헤더 컬럼과 `사진` 임베드 이미지를 자동 인식.
2. **시트 선택** — 상단 드롭다운(예: `Holex`, `EZ`).
3. **영역(품목) 선택** — 트리에서 행을 드래그/Shift로 범위 선택 후
   `선택영역 포함 / 제외`, 또는 `포함` 칸 클릭으로 개별 토글. `전체 선택/해제` 지원.
4. **순서 조정** — 행 선택 후 `▲ 위로 / ▼ 아래로`. 이 순서가 곧 슬라이드 배치 순서.
5. **PPT 설정** (우측)
   - 슬라이드당 품목 수 (최대 6)
   - 텍스트 내용: 품목명만 / +원산지 / +시즌
   - 품목명 글자 크기
   - 슬라이드 제목(시트명) 사용 여부
6. **💾 PPT 내보내기** — 저장 위치 지정 → 자동 생성.

## 고정 레이아웃 (템플릿 기준)

- 슬라이드: 27.5 × 19.1 cm
- 이미지: 약 5cm 정사각 슬롯(비율 유지·중앙 정렬)
- 텍스트박스: 이미지 바로 아래 고정
- 좌표는 `excel_to_ppt.py` 상단 `COL_X / ROW_Y / SLOTS` 에서 조정 가능.

## 구조

- `load_items(path, sheet)` — 품목명·이미지·원산지·시즌 추출 (GUI 없이 단독 호출 가능)
- `build_pptx(items, out_path, ...)` — 고정 위치 PPT 생성
- `run_gui()` — tkinter UI
