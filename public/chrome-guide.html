🧩 Chrome 확장 설치 가이드

AI 대화 수집기 — ChatGPT, Claude, Gemini 대화를 자동 추적합니다

1

Chrome 확장 관리 페이지 열기

Chrome 주소창에 아래 주소를 입력하세요:

chrome://extensions

또는 Chrome 메뉴 → 도구 더보기 → 확장 프로그램

2

개발자 모드 켜기

페이지 우측 상단의 "개발자 모드" 토글을 켜세요:

chrome://extensions
확장 프로그램
개발자 모드
3

"압축해제된 확장 프로그램을 로드합니다" 클릭

개발자 모드를 켜면 상단에 3개 버튼이 나타납니다. 왼쪽 첫 번째 버튼을 클릭:

4

chrome-extension 폴더 선택

파일 선택 창이 열리면, 설치된 Orbit 프로젝트의 chrome-extension 폴더를 선택:

📁 mindmap-viewer / chrome-extension

기본 경로:

~/mindmap-viewer/chrome-extension
⚠️ chrome-extension 폴더 자체를 선택하세요 (하위 파일이 아닌 폴더!)
5

설치 확인

성공하면 확장 목록에 "Orbit — AI 대화 수집기"가 나타납니다:

🛸
Orbit — AI 대화 수집기
v2.0.0 · ChatGPT, Claude, Gemini 대화 자동 수집
6

서버 연결 설정

Chrome 우측 상단 퍼즐 아이콘 → Orbit 확장 클릭 → 팝업에서 서버 URL 확인:

🛸 Orbit AI
AI 대화 자동 수집기
서버: sparkling-determination-production-c88b.up.railway.app
● 연결됨

✅ 설치 완료!
