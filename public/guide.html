화면별 사용법

각 뷰 모드 상세 가이드

개인, 팀, 부서, 회사 화면의 구체적인 사용법을 알아보세요.

👤 개인 화면

별자리 뷰

작업 세션들이 행성으로 표시됩니다. 각 행성은 하나의 프로젝트 또는 작업 세션을 나타냅니다.

프로젝트 클릭 → 펼침

행성을 클릭하면 해당 프로젝트의 파일 위성들이 펼쳐집니다. 파일 수정 횟수, AI 대화 내역 등을 확인할 수 있습니다.

위성 파일 노드

각 파일은 행성 주위를 도는 위성으로 표시됩니다. 파란색은 읽기, 주황색은 쓰기 작업을 나타냅니다.

실시간 추적

파일 편집, AI 대화, 터미널 명령이 실시간으로 3D 공간에 반영됩니다. 활성 작업에 글로우 효과가 나타납니다.

👥 팀 화면

팀원 배치

팀원들이 목표 구체를 중심으로 공전합니다. 각 팀원의 작업 진행률이 프로그레스 바로 표시됩니다.

실시간 협업

같은 파일을 동시에 수정 중인 팀원들은 연결선으로 표시되어 협업 관계를 한눈에 파악할 수 있습니다.

DM 채팅

팀원 노드를 클릭하면 정보 패널에서 직접 DM을 보낼 수 있습니다. 별도의 메신저 앱 필요 없이 바로 소통 가능합니다.

충돌 감지

두 팀원이 같은 파일을 동시에 수정하면 빨간색 경고가 표시됩니다. 충돌을 미리 방지할 수 있습니다.

🏛 부서 화면

부서별 그룹

각 부서가 하나의 행성계로 표시됩니다. 개발팀, 디자인팀, 기획팀 등이 각각의 궤도에 배치됩니다.

관리자 뷰

관리자는 부서별 진행률, 블로커, 마감일을 한눈에 확인할 수 있습니다. 관리자 패널에서 필터링도 가능합니다.

🏢 회사 화면

전사 현황

모든 부서의 행성계를 한 화면에서 조망합니다. 전체 조직의 업무 흐름과 병목을 파악할 수 있습니다.

KPI 대시보드

줌아웃하면 각 부서의 핵심 지표(완료율, 작업 수, 활동 시간)가 요약 칩으로 표시됩니다.

AI 모니터

멀티 AI 관제센터

Claude, GPT, Gemini, Grok 등 여러 AI를 하나의 모니터에서 동시에 관리하세요.

🖼

PIP 모드

3D 배경 위에 떠있는 창. 자유롭게 드래그, 리사이즈, 최소화. OBS 멀티소스 느낌.

⊞

타일 모드

화면 분할 그리드. CCTV 관제센터처럼 모든 에이전트를 동시에 모니터링.

▐

도킹 모드

좌=3D(70%) 우=스택(30%). 심플하고 안정적인 아코디언 토글.

📹
시작하기: 좌측 메뉴 → 📹 AI 모니터 클릭 → 에이전트 선택 → 레이아웃 모드 선택. 각 패널에는 상태 LED, 현재 작업, 최근 이벤트 스트림, 파일/메시지/활동 통계가 표시됩니다.

FAQ

궁금한 점이 있으신가요?

모든 데이터는 Railway 서버의 SQLite/PostgreSQL에 안전하게 저장됩니다. 사용자 인증은 JWT 토큰 방식이며, Google/GitHub OAuth를 지원합니다.
현재 샘플 데이터로 시뮬레이션되며, 향후 실제 Jira·GitHub·Slack 데이터 연동이 지원될 예정입니다.
왼쪽 사이드바 → 도구 → 🌍 행성 스킨을 클릭하면 구체/크리스탈/링/피라미드 등 8가지 모양을 선택할 수 있습니다. 선택한 스킨은 자동 저장됩니다.
로그인 후 자동으로 프로필 등록 안내가 표시됩니다. 이름·직함·회사·스킬·경력을 LinkedIn 수준으로 등록할 수 있으며, 팔로워들에게 공개됩니다.
현재는 데스크탑 브라우저에 최적화되어 있습니다. 터치 드래그로 기본 조작은 가능하지만, 정밀한 조작을 위해 데스크탑을 권장합니다.
