📋 주문 등록 자동화
현재 (수동) — 건당 2~3분
1. 카카오톡 거래처방에서 주문 메시지 확인
2. 품목명/수량 눈으로 파악
3. nenova ERP "신규 주문 등록" 화면 열기
4. 고객명 입력 → 품목 검색 → 수량 입력 → 저장
5. 다음 거래처 카톡방으로 이동
6. 하루 554회 반복
자동화 후 — 건당 ~3초
1. 카톡 화면 캡처 → Vision OCR 자동 읽기
2. 클립보드 감지 → 7종 파서 자동 구조화
3. 마스터 DB 매칭 (145품목+24거래처)
4. PAD/pyautogui가 nenova에 자동 입력
5. Vision 캡처로 입력 결과 자동 검증
6. 불일치 시 알림 → 수동 확인
사용 프로그램 + 기술
clipboard-watcher.js Vision OCR (Claude CLI) 마스터 DB (PostgreSQL) PAD (Power Automate Desktop) pyautogui (좌표 532건 학습) rawInput 검증
