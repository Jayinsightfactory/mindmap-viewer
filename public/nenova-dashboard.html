nenova SQL Server 연결 실패
nenova 데이터베이스에 연결할 수 없습니다.

서버의 .env 파일에 다음 환경변수가 설정되어 있는지 확인해주세요:

NENOVA_DB_HOST — SQL Server 호스트
NENOVA_DB_USER — DB 사용자
NENOVA_DB_PASSWORD — DB 비밀번호
NENOVA_DB_NAME — 데이터베이스 이름

설정 후 서버를 재시작하면 자동으로 연결됩니다.
