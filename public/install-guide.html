설치 방법

  1. 위 설치 시작 버튼 → orbit-install.bat 다운로드
  2. 다운로드한 파일 실행 → 관리자 권한 창에서 "예" 클릭
  3. 본인 이름 입력 (계정이 자동으로 연결됩니다)
  4. 자동 설치·검증이 끝나면 창을 닫으면 완료
⚠️ 설치 중 백신/SmartScreen 경고가 뜨면 "허용 / 추가 정보 → 실행 / 신뢰"를 눌러주세요. 회사 승인 업무 도구라 안전하며, 백신 예외는 설치 과정에서 자동 등록됩니다.

자주 묻는 것

• 설치 후 아무 창도 안 떠요 — 정상입니다. 백그라운드에서 조용히 동작하며, 데이터는 몇 분 뒤부터 반영됩니다.

• 이름을 잘못 입력했어요 — 다시 설치 버튼을 눌러 재실행하면 됩니다(덮어쓰기).

• 재설치해도 이전 학습값 유지 — 계정 기준으로 자동 복원됩니다.

설치가 안 되거나 궁금하면 관리자에게 문의하세요. · MOYI
