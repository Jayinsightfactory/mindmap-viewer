이용약관

시행일: 2026년 2월 28일  |  최종 수정: 2026년 2월 28일

이 약관은 Orbit AI(이하 "서비스") 이용자와 서비스 제공자 간의 권리, 의무 및 책임사항을 규정합니다.

제1조 — 서비스의 정의 및 목적

Orbit AI는 개발자의 AI 도구 사용을 실시간으로 추적하고 시각화하는 웹 기반 플랫폼입니다.

  • 핵심 기능: Claude Code, Cursor, Windsurf, VS Code 등 AI 도구 통합 추적
  • 시각화: 3D 신경망 맵, 생산성 분석, 인사이트 자동 생성
  • 데이터 저장: 기본 로컬 SQLite, 선택적 클라우드 동기화
  • 제공 언어: 한국어(기본), 다국어 지원

제2조 — 이용 조건

가. 회원가입

  • 만 18세 이상의 개인 또는 법인 사업자가 이용 가능
  • 회원가입 시 정확한 정보 입력 의무
  • 이메일 또는 소셜 로그인(Google, GitHub, Kakao, Naver, Apple) 지원
  • 중복 계정 생성 및 타인 명의 사용 금지

나. 계정 보안 책임

  • 회원은 자신의 계정 및 비밀번호 관리에 대한 책임 보유
  • 계정 도용 또는 비정상 접근 발견 시 즉시 서비스에 알릴 의무
  • 서비스는 명백한 과실 없이는 계정 관련 손해에 책임 제한

다. 서비스 이용 제한

  • 서비스 제공자 동의 없이 서비스 내용 복사, 수정, 배포 금지
  • 타인의 개인정보 수집 또는 부정행위 금지
  • 서버 과부하를 야기하는 대량 요청 또는 자동화된 접근 금지
  • 바이러스, 악성코드 등 유포 금지

제3조 — 금지 행위

회원은 다음 행위를 하면 안 됩니다:

  • 금지 타인을 사칭하거나 허위 정보 입력
  • 금지 개인정보 무단 수집 또는 3자 판매
  • 금지 서비스 원본 코드 재배포 또는 상업 목적 이용
  • 금지 불법 콘텐츠 생성 또는 유포
  • 금지 서비스 보안 취약점 악용
  • 금지 타인의 명예 훼손 또는 모욕
  • 금지 서비스 중단을 야기하는 행위

제4조 — 지적재산권

  • 서비스 소유권: Orbit AI 플랫폼, 로고, 콘텐츠는 서비스 제공자의 지적재산권
  • 회원 데이터: 회원이 생성한 AI 이벤트 데이터는 회원 소유
  • 라이선스 제한: 회원은 개인 용도로만 서비스 이용 권리 보유
  • DMCA: 저작권 침해 신고는 관리자 이메일로 연락
⚖️ 라이선스: Orbit AI는 MIT 라이선스로 배포되며, GitHub에서 오픈소스로 공개됩니다.

제5조 — 서비스 제공자의 면책

다음의 경우 서비스 제공자는 책임을 지지 않습니다:

  • 회원의 부주의 또는 자신의 계정 정보 유출로 인한 손해
  • 회원이 입력한 데이터 손실 (정기 백업은 회원 책임)
  • 제3자 서비스(OAuth, Ollama, Claude API 등)의 중단으로 인한 장애
  • 인터넷 연결 장애, 천재지변 등 불가항력적 사유
  • 서비스 개선을 위한 정기 유지보수 시간
⚠️ 보증 부인: 서비스는 "있는 그대로" 제공되며, 명시적 또는 묵시적 보증이 없습니다. 데이터 손실에 대해 서비스 제공자는 책임을 지지 않습니다.

제6조 — 약관의 변경

  • 서비스 제공자는 법적 요구, 서비스 개선 등의 이유로 약관을 변경할 수 있습니다
  • 변경 사항은 서비스 공지사항 또는 이메일로 공지합니다
  • 변경 후 서비스 계속 이용은 변경된 약관에 동의한 것으로 간주
  • 중대한 변경(요금 인상 등)의 경우 사전에 동의를 구할 수 있습니다

제7조 — 분쟁 해결

가. 분쟁 발생 절차

  • 서비스 관련 분쟁은 관리자 이메일로 문의
  • 관리자는 문의 접수 후 10일 이내 답변
  • 합의 불가능 시 중재 또는 소송 절차 진행 가능

나. 환불 정책

  • Orbit AI 기본 버전은 무료
  • 유료 기능(클라우드 동기화, API 등)에 대한 환불은 7일 이내 요청 시 전액 환불
  • 데이터 손실로 인한 손해배상은 지원하지 않음

제8조 — 준거법 및 관할

  • 이 약관은 대한민국 법을 준거법으로 합니다
  • 발생한 분쟁에 대한 관할권은 서울 중앙지방법원에 속합니다
  • 국제 분쟁의 경우 한-영 이중 언어 계약서가 우선
